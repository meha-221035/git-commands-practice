# Industry-Level Git & GitHub Commands Practice

## 1. Git Configuration Commands

### 1. git config --global user.name

### Command Name:
git config --global user.name

### Syntax:
git config --global user.name "Your Name"

### Purpose:
Sets the global username for Git commits.

### Example:
git config --global user.name "Mehanaz"



### 2. git config --global user.email

Command Name:
git config --global user.email

Syntax:
git config --global user.email "your-email@example.com"

Purpose:
Sets the global email address for Git commits.

Example:
git config --global user.email "mehanaz@example.com"
